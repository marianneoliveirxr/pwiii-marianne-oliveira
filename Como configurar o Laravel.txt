Como configurar o Laravel?

1 - Instale o XAMPP (http://sourceforge.net/projects/xampp/files/)
2 - Instale/ atualize por via das dúvidas o Node.js (https://nodejs.org/dist/v22.14.0/node-v22.14.0-x64.msi)
3 - Instale o Composer (https://getcomposer.org/Composer-Setup.exe) marcando a opção de escolha da versão do PHP (escolha a opção do XAMPP) e de sobrescrita do PATH.
4 - Siga as instruções de instalação do Laravel e criação do projeto (https://laravel.com/docs/11.x). Veja a partir do capitulo "Creating a Laravel Application".
5 - Entre na pasta do projeto pelo terminal
6 - Se você baixou seu projeto do GitHub execute o comando "composer install" seguido de "php artisan serve" para instalar as pastas necessárias (VENDOR e NODE_MODULES) e iniciar o servidor.
